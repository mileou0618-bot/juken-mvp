# 交付检查清单（中文）

每一单在发 PDF 之前，先快速检查下面这些点。

## 1. 诊断信息是否正确

- [ ] diagnosis_id 是否正确
- [ ] 家长提供的诊断ID和当前订单一致
- [ ] followup_answers 是否完整

## 2. 整理包内容是否生成

- [ ] weekly_focus 是否生成
- [ ] 主要问题是否有内容
- [ ] 保住 / 减少 / 暂缓 是否有内容
- [ ] 家长要做什么 / 不要做什么 是否有内容
- [ ] 孩子本周只练什么 是否有内容
- [ ] 7天行动安排是否有内容

## 3. 文件是否可用

- [ ] Docs 链接是否生成
- [ ] PDF 是否可打开
- [ ] PDF 内容是否和 Docs 一致
- [ ] 中文是否正常显示

## 4. 人工确认

- [ ] 已经人工快速看过一遍内容
- [ ] 这份整理包没有明显跑题
- [ ] 主题没有明显模板化错误

## 5. 收款状态

- [ ] 是否已收款
- [ ] payment_status 是否已标记为 paid

## 6. 发出状态

- [ ] 是否已发送
- [ ] 发送给的诊断ID是否正确
- [ ] 对方是否已确认收到

## 7. 简单判断

如果以上大部分都没问题，就可以发送。

如果某一步不确定，先停一下再发，不要急着交付。
