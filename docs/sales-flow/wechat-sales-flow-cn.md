# 微信成交话术（中文）

适用场景：用户在微信发送“家庭学习整理 / 诊断ID：JUKEN-XXXX”后，人工回复使用。

## 1. 第一条回复

你好，感谢你发来诊断ID。

我先看一下这次诊断结果和补充信息，再帮你整理这周最该保住什么、先减掉什么。

这份服务是根据诊断结果 + 追加6问整理出来的《7天家庭学习整理包》，主要是帮家庭先把本周的学习顺序理顺。

## 2. 1980日元服务说明

这份服务费用是 1980 日元。

它不是成绩提升承诺，也不是个别补课，而是把你家这一周最该处理的内容整理清楚：

- 本周先保住什么
- 本周先减少什么
- 家长这一周该做什么
- 孩子这一周只练什么
- 接下来7天怎么安排

如果你觉得现在最需要的不是“再加内容”，而是“先把顺序理清”，这份整理包会更合适。

## 3. 交付内容说明

购买后会交付一份 PDF，里面会包含：

- 当前问题结构
- 本周必须保住的内容
- 本周应该减少的内容
- 家长本周做什么 / 不做什么
- 孩子本周只练什么
- 7天行动安排
- 本周观察重点

如果有必要，也可以先看一下这周最该守住的两个点，再决定怎么推进。

## 4. 支付前确认话术

如果你愿意，我可以先帮你确认这份整理包是否适合你家。

你只要把：
- 诊断ID
- 孩子年级
- 现在最困扰的一件事

发给我，我先看一下再告诉你是否适合购买。

## 5. 收款后确认话术

收到后我会开始整理。

整理好以后，我会把 PDF 发给你。

## 6. 交付 PDF 时话术

这是你家的《7天家庭学习整理包》PDF。

你可以先看：
- 本周保什么
- 本周减什么
- 家长要做什么
- 孩子只练什么

如果你看完后想再确认下一步怎么推进，也可以继续发我消息。

如果你现在更需要的是：

那这份整理包可能不太适合。

它更适合已经有诊断结果，希望先把家庭学习顺序理一理的情况。
