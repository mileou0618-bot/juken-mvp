# 收款方式说明（中文）

## 1. 前期可用支付方式

前期优先使用简单、可人工确认的方式：

- PayPay 个人收款
- 银行转账

如果后续需要再扩展别的方式，也可以再加。

## 2. PayPay 个人收款话术

可以这样说：

> 这份整理包费用是 1980 日元。
> 如果方便的话，可以通过 PayPay 付款。
> 付款后请把截图发给我，我收到后就开始整理。

## 3. 银行转账话术

可以这样说：

> 也可以用银行转账。
> 付款完成后，请把转账截图发给我，我这边确认后就开始整理。

## 4. 收款记录方式

前期建议人工记录在表格里，字段可以先简单一点：

- diagnosis_id
- 家长昵称 / 微信名
- 付款方式
- 金额
- 付款时间
- payment_status
- 备注

payment_status 建议先用这些值：

- pending：待付款
- paid：已付款
- refunded：已退款（如后续需要）

## 5. 付款确认后如何标记 payment_status

确认收到款后，把该单标记为：

- `payment_status = paid`

如果还没收到款，就保持：

- `payment_status = pending`

如果后面人工退回，再标记为：

- `payment_status = refunded`

## 6. 前期操作建议

前期不需要复杂系统。

只要做到：

1. 看见诊断ID
2. 确认付款
3. 标记状态
4. 发 PDF

就足够支撑前几单的交付了。
