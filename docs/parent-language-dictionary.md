# 中学受験｜家庭学習管理「家长语言辞典 v1」

把系统内部的风险维度、分析语言、咨询语言，翻译成日本家长真实能感知、能代入的表达。

## 1. 宿題負荷（homework_load）

### 系统语言
- 宿題負荷
- 学習負荷
- 処理負荷
- 宿題量

### 家长真实感知
- 毎日こなすだけで終わっている
- 解き直しまで手が回らない
- 宿題を終わらせるだけで精一杯
- 「終わった？」ばかりになっている
- 量に追われている感じがする
- 毎日バタバタしている

### 情绪感知
- 余裕がない
- 回しているだけ
- 常に追われている
- 毎日ギリギリ

### 危险状态
- 睡眠を削っている
- テスト前だけ崩れる
- 終わらないまま次へ進む

---

## 2. 復習・定着不足（review_retention）

### 系统语言
- 復習不足
- 定着不足
- 理解不足

### 家长真实感知
- 同じミスを何回もする
- 前にやったはずなのに忘れている
- 「分かった」が続かない
- 解き直しても定着していない
- 点数につながらない
- やった量の割に伸びない

### 情绪感知
- やりっぱなし
- 積み上がっていない
- 空回りしている感じ

### 危险状态
- 解答を写して終わる
- 理解より提出優先
- 「できたつもり」が増える

## 使用原则

1. 对用户展示时，优先使用「家长真实感知」里的表达。
2. 系统内部可以使用 risk dimension，但用户侧文案必须翻译成人能感知的语言。
3. 不要直接把「親の関与過多」「精神的負荷」「計画・優先順位」等分析词作为主标题。
4. Hero、LINE、LP、广告优先使用具体家庭场景。
5. 目标是让用户读到后觉得：「これはうちのことだ」。
6. 禁止责备家长或孩子。
7. 禁止使用成绩保证、合格保证、手遅れ、危険等煽り表达。
8. 文案应更像教育咨询中的现状整理，而不是AI分析报告。
9. 表达要具体、短、可感知。
10. 后续所有结果页、LINE、LP、广告文案修改，都应先参照本辞典。
