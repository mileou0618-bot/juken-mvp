# 家长放手

## 输入
- weekly_focus: 家长放手
- grade_stage: 小学4年级
- study_end_time: 20-21点
- hardest_subject: 没有特别偏科
- current_main_problem: 家长不敢放手
- sacrificed_area: 亲子关系的余裕

## one_line_conclusion
本周重点不是家长继续加力，而是减少临时介入，让孩子先承担一小部分判断。

## surface_problem
家长不敢放手

## real_problem
不是家长不够认真，而是家庭学习已经过度依赖家长提醒，孩子自己判断和启动的机会太少。

## 保住
- 塾明确要求提交的内容
- 孩子自己说最不会的1〜2个地方
- 每天一个固定开始动作

## 减少
- 家长临时追加的任务
- 反复催促“快点做”
- 家长替孩子重新安排全部计划

## 暂缓
- 家长临时改计划
- 当天追加新要求
- 把所有任务重新排一遍

## 家长目标
家长本周的目标不是管得更细，而是把一部分判断交还给孩子。

## 孩子目标
孩子本周的目标不是完全自觉，而是每天能自己说出第一件要做的事。

## 家长做
- 每天只确认“第一件事是什么”
- 让孩子自己说出最卡的地方

## 家长不要做
- 不要从头管到尾
- 不要孩子一慢就马上接管

## 孩子本周只练
每天自己说出第一件要做的事，并完成它

## 今晚
今晚让孩子自己说出明天第一件要做的事。

## 明天
明天只确认第一件事有没有开始，不立刻追加其他要求。

## 这一周
这一周每天少管一个环节，让孩子自己做一次选择。

## 周末
周末和孩子一起确认：下周哪一件事可以由孩子自己决定。

## 观察重点
- 孩子是否能说出第一件要做的事
- 家长催促次数是否减少
- 孩子是否有一次自己开始学习

## 7天执行表
- Day1：先让孩子自己说出明天第一件要做的事。｜孩子今天有没有先说出第一件事？
- Day2：只确认第一件事有没有开始，不立刻追加。｜今天有没有少催一点？
- Day3：每天少管一个环节，让孩子自己做一次选择。｜孩子有没有自己做一次选择？
- Day4：把“下一步做什么”交给孩子说出来。｜孩子能不能自己说下一步？
- Day5：看看家长催促的次数有没有减少。｜今天有没有少接管一次？
- Day6：让孩子自己开始一次，不从头管到尾。｜今天有没有让孩子自己开始？
- Day7：和孩子一起确认下周哪一件事可以自己决定。｜下周哪一件事可以交给孩子决定？
